Aviso de Privacidad

ContaVida, antes establecido como ngLifeCounter (en adelante, “el Proyecto”) es un proyecto de carácter tecnológico que respeta y protege los datos personales de sus usuarios, en cumplimiento de la Ley Federal de Protección de Datos Personales en Posesión de los Particulares (LFPDPPP) y demás normatividad aplicable.

Al registrarte y utilizar este sitio web, aceptas de manera expresa los términos establecidos en el presente Aviso de Privacidad.

Responsable del tratamiento de los datos

El responsable del tratamiento de los datos personales recabados a través de este sitio es el desarrollador y administrador del Proyecto ContaVida.

Información que se recopila

Para el correcto funcionamiento del Proyecto, únicamente se recopila la siguiente información personal:

  • Nombre
  • Apellido
  • Correo electrónico (utilizado exclusivamente como medio de contacto y autenticación), en caso de autenticación con Google o terceros JAMAS se guardan contraseñas de dichos correos
  • Dirección IP (utilizada para fines de seguridad, prevención de fraudes y ciberataques)

Finalidad del tratamiento de los datos

Los datos personales recabados serán utilizados únicamente para:

  • Permitir el acceso y uso de la plataforma
  • Identificación y autenticación del usuario
  • Comunicación relacionada con el funcionamiento del servicio
  • Prevención de accesos no autorizados, fraudes o ataques informáticos

Almacenamiento y uso de servicios en la nube

La información personal es almacenada y procesada utilizando servicios de infraestructura en la nube proporcionados por terceros, incluyendo pero no limitándose a:

  • Microsoft Azure
  • MonsterASP

Ambientes de producción, desarrollo y pruebas

El Proyecto cuenta con distintos ambientes tecnológicos, incluyendo ambientes de producción, desarrollo y pruebas.

La confidencialidad, integridad y resguardo adecuado de los datos personales solo está garantizada cuando el usuario final accede y utiliza exclusivamente el sitio de producción oficial del Proyecto.

Los ambientes de desarrollo y pruebas tienen como finalidad la validación técnica del sistema y pueden exponer públicamente configuraciones, estructuras de base de datos o código (Por su naturaleza Open Source), por lo que no están diseñados ni destinados para el ingreso de datos personales reales.

En caso de que el usuario introduzca voluntariamente información personal en ambientes de desarrollo o pruebas, lo hace bajo su propia responsabilidad, liberando al Proyecto y a su administrador de cualquier daño, pérdida, uso indebido, exposición o incidente de seguridad relacionado con dichos datos.

Pagos, cobros y prevención de phishing

ContaVida NO procesa, NO solicita y NO administra pagos de ningún tipo, bajo ninguna circunstancia.

El Proyecto jamás solicitará información financiera, incluyendo pero no limitándose a: números de tarjetas bancarias, cuentas, CLABE, contraseñas, códigos de verificación, transferencias, depósitos o cualquier otro medio de pago.

Cualquier sitio, correo electrónico, mensaje, enlace o comunicación que solicite pagos o información financiera haciéndose pasar por ContaVida debe considerarse un intento de fraude o phishing y no está relacionado de ninguna manera con el Proyecto ni con su administrador.

El Proyecto no se hace responsable por daños derivados de interacciones del usuario con sitios o terceros que suplanten o imiten la identidad de ContaVida.

Seguridad de la información

El Proyecto implementa medidas razonables de seguridad para proteger los datos personales contra daño, pérdida, alteración, destrucción o uso, acceso o tratamiento no autorizado. No obstante, el usuario reconoce que ningún sistema es completamente seguro y acepta dicho riesgo.

Transferencia de datos

No se venden, ceden ni transfieren datos personales a terceros, salvo aquellos casos necesarios para la operación técnica del Proyecto o cuando exista una obligación legal que así lo requiera.

Derechos ARCO

El usuario podrá ejercer en cualquier momento sus derechos de Acceso, Rectificación, Cancelación u Oposición (ARCO) respecto a sus datos personales, mediante solicitud expresa dirigida al administrador del Proyecto.

Proyecto Open Source

ContaVida es un proyecto de código abierto (open source). El código fuente se encuentra disponible públicamente en GitHub: https://github.com/OpenSourceRvProjects/ngLifeCounter.Code .

La disponibilidad del código fuente no implica la divulgación de datos personales de los usuarios.

Aceptación del aviso de privacidad

Al registrarse y utilizar este sitio, el usuario manifiesta haber leído, comprendido y aceptado el presente Aviso de Privacidad.
